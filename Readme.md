# Talk Through Time 🎴⌛

A browser-based flashcards study app that transforms modern phrases into classic language through interactive decks and cards. Built as a single-page application using HTML, CSS, and JavaScript with LocalStorage for persistent data.

## 📌 Project Overview

Talk Through Time is a responsive flashcards app where users can create decks, add cards, study content, and flip between the modern phrase and its classic translation.

This project was built to demonstrate front-end development fundamentals, DOM manipulation, state management, accessibility, and AI-assisted development workflows.

---

## ✨ Features

### Deck Management
- Create new decks
- Edit deck names
- Delete decks
- Switch between multiple decks

### Card Management
- Add new flashcards
- Edit existing cards
- Delete cards

### Study Mode
- Flip cards front ↔ back
- Previous / Next navigation
- Shuffle cards
- Keyboard shortcuts

### Search & Persistence
- Search cards by keyword
- Saves decks/cards with LocalStorage
- Restores previous session on reload

### Accessibility & UI
- Responsive layout for mobile and desktop
- Keyboard navigation support
- Focus states
- Modal dialogs
- Clean themed interface
